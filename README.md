# Stories

Story Reader app Assignment for Pratilipi using Django and Angular

## Development
  - Clone the repository
### Backend
  Change directory to repo folder.
  Install dependencies:
  `pip3 install -r requirements.txt`
  
  Run server:
  `python3 manage.py runserver`

### Frontend
   Install dependencies:
  `npm install`
  
  Run server:
  `ng serve`
